# RotiBank 🍽️

## 🔗 Live Project  
👉 https://rotibank1.vercel.app/

## 💡 Overview  
RotiBank is a social-impact-driven web platform designed to reduce food wastage while addressing hunger.

It connects food donors (restaurants, households, events) with NGOs and volunteers, ensuring surplus food reaches those in need efficiently.

## ❗ Problem Statement  
- Large-scale food wastage despite existing hunger  
- Lack of real-time coordination between donors and NGOs  
- No fast and reliable system to distribute surplus food  

## ✅ Solution  
RotiBank provides a simple digital platform that:  
- Enables donors to list surplus food quickly  
- Allows NGOs/volunteers to access real-time availability  
- Facilitates fast coordination for timely distribution  

## 🛠️ Tech Stack  
- Frontend Web Application  
- Cloud Deployment (Vercel)  
- Focus on clean and intuitive UI  

## 🌍 Impact  
- Reduces food wastage  
- Improves food accessibility  
- Encourages community participation
